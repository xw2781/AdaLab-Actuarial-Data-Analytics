ArcRho Python API User Guide

Use arcrho_api to read and edit ArcRho projects, reserving classes, and DFM methods from Python notebooks or scripts.

Install

ArcRho release builds include a wheel that can be installed with pip.

python -m pip install "<ArcRho install folder>\resources\python_packages\arcrho_api-0.1.0-py3-none-any.whl"

For development from this repository:

python python-api\tools\build_wheel.py --out-dir python-api\dist
python -m pip install python-api\dist\arcrho_api-0.1.0-py3-none-any.whl

Server Root

By default, ArcRhoClient() reads the same server root used by the ArcRho host app: %APPDATA%\ArcRho\workspace_paths.json. Set it in the ArcRho app through Server Connection, or update the same config from Python. 

from arcrho_api import ArcRhoClient, get_server_root, set_server_root

print(get_server_root())

set_server_root(r"E:\ArcRho Server")
client = ArcRhoClient()
Passing ArcRhoClient(r"E:\ArcRho Server") still works for one script without changing the saved host config.

Basic Workflow

from arcrho_api import ArcRhoClient

client = ArcRhoClient()
print(client.list_projects())

project = client.project("Current Reserve Review")
rc = project.reserving_class(r"Auto\Private Passenger")
dfm = rc.dfm("Paid Loss Ultimate")

print(dfm.name)
print(dfm.input_triangle)
print(dfm.output_vector)
print(dfm.notes)

Edit A DFM

DFM objects are read/write by default, but changes are only written to disk when save() is called.

dfm.clear()
dfm.exclude_covid_years()
dfm.exclude_high(dev_period=2, count=1, reason="High selected ratio reviewed.")
dfm.set_selected_average("Simple - 3")
dfm.add_notes("Updated through arcrho_api.")

dfm.save()

Create A DFM

new_dfm = rc.new_dfm(
    "New Paid Loss Ultimate",
    output_vector="New Paid Loss Ultimate",
    input_triangle="Paid Loss",
    origin_length=12,
    development_length=12,
)
new_dfm.save()

Read-Only Mode

Use read-only mode for review notebooks where accidental writes should be blocked.

client = ArcRhoClient(read_only=True)
dfm = client.project("Current Reserve Review").reserving_class(r"Auto\Private Passenger").dfm("Paid Loss Ultimate")

dfm.add_notes("This will stay in memory.")
dfm.save()  # raises ReadOnlyError

Legacy Notebook Style

ArcRhoSession provides a context-bound wrapper for legacy reserve-review notebooks.

from arcrho_api.migration import ArcRhoSession

session = ArcRhoSession()
session.set_project("Current Reserve Review")
session.set_reserving_class(r"Auto\Private Passenger")

aDFM = session.DFM("Paid Loss Ultimate")
aDFM.clear()
aDFM.ex_COVID_AY()
aDFM.set_selected_estimate("Simple - 3")
aDFM.save()
